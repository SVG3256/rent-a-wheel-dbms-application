# Rent-A-Wheel Car Rental Management System
### Modern DBMS application to manage hassle-free car rentals
---
Car rentals are the go-to solution for easy commuting in the city. However, renting a car can often be a cumbersome process, involving phone calls or inflexible in-person visits. ```RentAWheel``` is a web application designed to solve this by creating a centralized, real-time system. It is aimed at providing a seamless interface for customers to browse, book, and manage their car rentals online.
